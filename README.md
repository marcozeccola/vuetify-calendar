# vuetify-calendar

Simple CRUD calendar sistem with vue.js vuetify material framework and firebase.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
